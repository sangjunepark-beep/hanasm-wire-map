깃헙 토큰 입력

개인용 접근 토큰(PAT)을 입력하면 비공개 리포까지 실시간으로 불러옵니다. 토큰은 이 브라우저에만 저장되고 파일·서버에 올라가지 않습니다.
· 토큰 발급: github.com/settings/tokens → repo 권한(비공개 포함) 또는 fine-grained 토큰의 Contents/Metadata 읽기 권한.
· 토큰 만료 시 여기서 새 토큰만 다시 붙여넣으면 됩니다.
